# COVID-19 Prediction

This project uses Machine Learning to predict whether a person is likely to test positive or negative for COVID-19 based on the given symptoms and health-related data.

## Objective

To build a machine learning model that can predict COVID-19 infection status.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation
6. COVID-19 Prediction

## Result

The trained machine learning model predicts the COVID-19 status based on the input data.
